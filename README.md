# Trading BTC using Deep Learning
This is a research project, To Analysis the cryptocurrencies price using different Machine learning Algorithm.

# Run the project

```bash
./setup_demo.sh
```

```bash
airflow webserver -p 8080
```

```bash
airflow scheduler
```

```bash
cd notebooks
jupyter notebook
```
